# twitter2nitter

Redirects your Twitter/X page to Nitter, that's it. Forked from a-catgirl-dev's version, 
but this one strips the tracking data from the twitter url since sometimes nitter.net 
chokes on it a little bit. Does what it says on the tin. 

You can download the extension [here](https://github.com/goloins/twitter2nitter/releases/download/release-verified/twitter2nitter.signed.xpi)
